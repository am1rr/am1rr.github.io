Instalar Telnet

Lo primero que tenemos que hacer es instalar telnet mediante "sudo apt install telnetd"

Despues ejecutamos el siguiente comando para iniciar el servidor Telnet

Comprobamos que el Telnet esta funcionando con un "nmap localhost"

Cuando lo tengamos ejecutado nos conoectaremos con el comando "telnet 192.168.18.210" Esta última es la Ip de mi RaspberryPi

Probamos comandos y vamos al WireShark y desde allí buscamos paquetes Telnet y seleccionamos la opcion "Follow TCP Stream"
