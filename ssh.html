Que es SSH?

SSH es el nombre de un protocolo y del programa que lo implementa cuya principal función es el acces o remoto a un servidor por medio de un canal seguro en el que toda la información está cifrada.

La Practica 1 consiste en conectarse mediante telnet a nuestra Raspberry Pi y ejecutar algunos comandos, luego lanzar el wireshark y comprobar que los datos no van encriptados

PASO 1

Instalar Telnet mediante el siguiente comando

sudo apt install telnetd

PASO 2

Despues ejecutamos el siguiente comando para iniciar el servidor Telnet

sudo /etc/init.d/openbsd-inetd restart

PASO 3

Comprobamos que el Telnet esta funcionando con un:

nmap localhost

PASO 4

Cuando lo tengamos ejecutado nos conoectaremos con el comando (192.168.18.210 es la Ip de la Raspberry):

telnet 192.168.18.210

PASO 5

Ejecutamos algunos comandos desde Telnet y vamos al wireshark a analizar trafico.

Follow TCP Stream

FIN

En Wireshark podemos observar como los comandos van en texto plano y no estran encriptados. Y ya queda concluida la primera practica de SSH.
